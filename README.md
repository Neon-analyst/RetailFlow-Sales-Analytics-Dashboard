# RetailFlow-Sales-Analytics-Dashboard
End-to-end retail analytics project: data cleaning, modeling, and dashboard visualization.

A comprehensive analysis and visualization of retail sales performance using interactive dashboards.


## Project Description
The **RetailFlow Sales Analytics Dashboard** is a data-driven project designed to analyze and visualize retail sales trends, customer behavior, and managerial performance.

This project highlights skills in **data cleaning, modeling, dashboard , business intelligence, and storytelling with data**.

The dashboard provides insights into:

- Total revenue  
- Average sales value per transaction  
- Manager performance  
- Highest ranking  entries
- Community-level market behavior  
- Peak selling months and days  

##  Dashboard Overview
The dashboard features multiple interactive visuals:

- **Total Sum of Sales**
- **Average Sales Value**
- **Manager Headcount**
- **Unique Buyers**
- **Top 10 Performing Managers**
- **Highest-Ranking Products/Categories**
- **Top Communities**
- **Highest ranking entries**
- **Highest Revenue Days**
- **5 Highest-Selling Months**
- Interactive slicers for **Year** and **Financial Year**

Designed with a modern dark theme for readability and professional presentation.



##  Data Cleaning & Transformation

Data cleaning was performed using ** Excel Power Query** and includes:

- Removing duplicates  
- Handling missing values  
- Normalizing text/categorical fields  
- Creating calculated columns  
- Building relationships between tables  
- Time intelligence preparation  


## Key Insights

- RetailFlow generated **over $19.1M** in sales.  
- The **average sale value** is approximately **$835**.  
- The business operates efficiently with **21 managers**.  
- Over **22,865 unique buyers** were recorded.  
- Several managers significantly outperform others.  
- **Monday** and **Friday** are top revenue days.  
- **July** and **November** are peak-selling month.
